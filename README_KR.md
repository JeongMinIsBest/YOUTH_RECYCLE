# 랜덤 포레스트 및 전주시 데이터 분석 기반 전주시 쓰레기 문제 해결 방안 제시 🚮
> 제2회 전북 청년 빅데이터 경진대회 – *우리은행 은행장상* 수상작(2024.12)
  
📊 주최 : 전북 빅데이터 기술교류 공동연구회, 우석대학교 링크3.0 사업단, 전북대학교 빅데이터 혁신융합대학 사업단
  
👩‍💻 기획 및 분석 : 강민수, 이기백, 이혜원, 임정민
  
🗓 프로젝트 기간 : 2024.10.01 ~ 2024.10.04
<br/>
<br/>

## 1. 프로젝트 개요
전주시 원룸촌 및 주택가에서 지속적으로 발생하는 생활폐기물 분리배출 문제와 무단 투기 문제를 해결하기 위해 
**행정동 단위 공공데이터 분석과 머신러닝 모델링을 결합하여 쓰레기 배출 문제가 가장 빈번하게 발생하는 구역을 찾고,
‘재활용 정거장(Recycling Station)’ 정책 제안 및 시행 최적 입지를 선정**하는 데이터 기반 정책 제안 프로젝트입니다.
  
본 프로젝트는 단순 예측이나 시각화에 그치지 않고,
문제 발생의 **구조적 원인 진단 → 고위험 지역 선별 → 정책 개입 지점 제안**이라는  
정책 의사결정 흐름을 데이터로 구현하는 것을 목표로 합니다.
<br/>
<br/>

## 2. 연구 배경 및 문제의식
전주시는 원룸촌 · 주택가를 중심으로 다음과 같은 문제가 반복적으로 제기되어 왔습니다.
  
- 생활폐기물 불법 투기의 지속적 증가  
- 분리배출 장소 부족 및 규정 미준수  
- 요일·시간 제한 중심 정책으로 인한 주민 불편  
- CCTV·안내문 중심 기존 대책의 한계  
  
이는 단순 시민 의식의 문제가 아니라, **인구 밀도·주거 형태·지역 구조가 결합된 도시 환경 문제**로 볼 수 있습니다.
  
본 프로젝트는 다음 질문에서 출발했습니다.
  
- 왜 특정 행정동에서 쓰레기 문제가 반복되는가?  
- 인구 규모 외에 어떤 구조적 요인이 작용하는가?  
- 데이터 기반 분석은 정책 우선순위 설정에 어떻게 기여할 수 있는가?  
<br/>
<br/>

## 3. 데이터 구성
- **분석 단위**: 전주시 행정동
  
- **데이터 출처**
  - 전주시 인구 현황 데이터
  - 주택·다세대·오피스텔 분포 데이터
  - 전주시 쓰레기 배출량 데이터
  - 쓰레기 불법투기 단속 현황
  - 행정동 경계(GEOJSON)
  
- **주요 변수**
  - 인구 수 및 인구 밀집도
  - 원룸촌·주택가 밀집도
  - 쓰레기 배출량 및 문제 발생 빈도
  
- **전처리**
  - 행정동 기준 데이터 통합
  - 결측치 처리
  - Min–Max Scaling을 통한 변수 정규화
<br/>
<br/>

## 4. 분석 방법론

### 4.1 탐색적 데이터 분석 (EDA)
- 행정동별 인구·주거·쓰레기 분포 시각화
- 변수 간 상관관계 히트맵 분석
- 공간적 편중 및 문제 집중 지역 탐색
<br/>

### 4.2 머신러닝 기반 예측
- **목적**: 전주시 행정동별 연간 쓰레기 배출량 추정
  
- **모델**
  - Random Forest Regressor
  - (보조 비교 모델) MLP
  
- **모델 선택 이유**
  - 비선형 관계 학습 가능
  - 이상치 및 복합 변수 환경에 강건
  - Feature Importance 기반 해석 가능성
<br/>

### 4.3 클러스터링 분석
- K-means, K-medoids, Hierarchical, GMM 적용
- 인구 밀집도 · 주거 밀집도 · 쓰레기 배출량 기반 지역 유형화
- 고위험 클러스터 우선 선별
<br/>
<br/>

## 5. 기대 효과
- 전주시 불법 폐기물 투기 감소
- 쓰레기 관련 민원 감소 및 도시 미관 개선
- 재활용률 증가 및 수거 비용 절감 가능성
- 지역 사회 일자리 창출 및 시민 참여 강화
<br/>

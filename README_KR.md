## 1. 프로젝트 개요
전라북도 전주시를 대상으로 행정동 단위의 생활폐기물 배출량을 분석하고,  
머신러닝 기반 예측 모델을 활용하여 전주시 쓰레기 문제 해결 방안을 제시합니다.
  
단순한 현황 분석을 넘어, 생활폐기물 문제가 **인구 규모가 아닌 구조적 · 공간적 요인**에서 발생함을 규명하고  
이를 정책적으로 활용할 수 있는 분석 프레임을 제시하는 것을 목표로 합니다.
<br/>
<br/>

## 2. 연구 동기 및 문제의식
도시의 폐기물 문제는 단기적인 수거 효율의 문제가 아니라,  
인구 밀도, 주거 형태, 지역 구조가 복합적으로 작용하는 문제입니다.
  
그러나 기존 정책은 종종 사후적 대응이나 일괄적인 규제에 머무르며,  
지역 간 구조적 차이를 충분히 반영하지 못하는 한계를 보입니다.
  
본 프로젝트는 다음과 같은 질문에서 출발했습니다.
  
- 왜 특정 지역에서 폐기물 문제가 반복적으로 발생하는가?  
- 인구 수 외에 어떤 요인이 폐기물 배출량에 영향을 미치는가?  
- 예측 모델은 정책 우선순위 설정에 어떻게 기여할 수 있는가?  
<br/>
<br/>

## 3. 데이터 설명

- **데이터 출처** : 전주시 공공데이터, 통계청(KOSIS)
- **분석 단위** : 행정동
- **주요 변수**
  - 인구 수 및 인구 밀도
  - 주거·주택 관련 지표
  - 환경 및 폐기물 관련 지표
- **전처리 과정**
  - 행정동 기준 데이터 통합
  - 결측치 처리
  - 변수 스케일 정합 및 정규화
<br/>
<br/>

## 4. 분석 방법론

### 4.1 탐색적 데이터 분석 (EDA)
- 지역별 폐기물 배출량 분포 분석
- 인구·주거 특성과 폐기물 배출량 간 상관관계 분석
- 공간적 편중 현상 시각화
<br/>

### 4.2 예측 모델링
- **모델** : Random Forest Regressor
- **선택 이유**
  - 비선형 관계 학습 가능
  - 다중 변수 환경에서 안정적인 성능
  - 변수 중요도 분석을 통한 해석 가능성 확보
- **모델 평가**
  - 교차검증(Cross-validation)
  - RMSE, MAE 기반 성능 평가
  - Feature Importance 분석
<br/>
<br/>

## 5. 주요 분석 결과
- 생활폐기물 배출량은 인구 규모만으로 설명되지 않음
- 주거 밀도 및 지역 구조 변수가 유의미한 영향
- 예측 모델을 통해 **구조적 위험 지역 식별 가능성** 확인
- 변수 중요도 분석 결과는 정책 설계 시 실질적 근거 제공
